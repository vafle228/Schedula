**Schedula** — десктопное приложение для автоматизации составления учебных планов и расписаний. Продукт превращает сложный хаотичный процесс распределения академической нагрузки из многостраничных Excel-таблиц в наглядную, быструю и контролируемую работу с интерактивной сеткой.

Интерфейс спроектирован по принципу высокой информационной плотности в духе современных SaaS-инструментов (Notion, Linear), обеспечивая комфортную многочасовую работу методиста без когнитивной перегрузки.

---

## 🚀 Ключевые возможности

### 1. Модуль «Распределение» (Учебный план)
*   **Управление нагрузкой:** Удобный пул карточек дисциплин с мгновенным визуальным разделением на теорию и практику.
*   **Интерактивный Drag-and-Drop:** Закрепление дисциплин за преподавателями путем перетаскивания карточки на строку сотрудника.
*   **Умные индикаторы:** Живой подсчет суммарных часов нагрузки преподавателя в реальном времени с мягким предупреждением о превышении условной нормы.
*   **Поддержка связи «многие-ко-многим»:** Возможность назначения одной дисциплины нескольким преподавателям.

### 2. Модуль «Расписание»
*   **Автогенерация:** Встроенный движок для автоматического построения базовой сетки недели на основе ограничений, доступности преподавателей и вместимости аудиторий.
*   **Умная Master-Сетка:** Гибкое представление расписания по группам с возможностью тонкой ручной доводки (перетаскивание, закрепление/Pin удачных слотов).
*   **Постоянный контроль конфликтов:** Выделенная панель живого лога, которая мгновенно подсвечивает накладки (один преподаватель в двух местах, занятая аудитория, превышение лимита пар).
*   **Обработка «осиротевших» пар:** Если назначение в плане изменилось, система не удаляет пару молча, а маркирует её как конфликтную для ручного переназначения.

### 3. Экспорт и импорт
*   Выгрузка готовых результатов распределения нагрузки в Excel строго по государственному шаблону.
*   Гибкий экспорт расписания по группам или преподавателям с предварительной валидацией критических ошибок перед генерацией файла.

## 🛠 Технологический стек

Приложение разработано на стыке производительного бэкенда и легковесного десктопного окружения:
*   **Backend:** Python (бизнес-логика, алгоритмы автоматической генерации расписания, парсинг и сборка Excel-документов).
*   **Frontend:** Single Page Application (SPA) на JavaScript (интенсивный интерактивный UI, drag-and-drop паттерны, виртуализированные списки для плавной работы со 100+ элементами).
*   **Runtime:** Десктопная упаковочная среда для локального запуска веб-технологий.

---

## 🎹 Десктопные паттерны и Доступность (WCAG 2.1 AA)

*   **Клавиатурный флоу:** Полная альтернатива drag-and-drop операциям. Назначение и расстановка элементов доступны через контекстные меню, горячие клавиши и быстрый поиск по нажатию пробела.
*   **Ресайз панелей:** Настраиваемая двух- и трехпанельная компоновка главного экрана.
*   **Надежность:** Обязательная поддержка истории операций (`Undo` / `Redo`) на всех этапах ручной доводки расписания.
*   **Визуальная доступность:** Типы дисциплин и конфликтные состояния кодируются не только цветом, но и иконками/текстовыми бейджами для удобства восприятия.
